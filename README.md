# Snell's Law Web App

A simple web app to calculate refraction angles using Snell's Law and known refractive indices of common materials.

## Features
- Calculates refracted angle for user-selected materials
- Handles total internal reflection
- Includes refractive indices for air, water, glass, silicon, diamond, and plastic

## Usage
1. Open `index.html` in your browser.
2. Select materials and enter the incident angle.
3. Click Calculate to see the refracted angle or total internal reflection.
